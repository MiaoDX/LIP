## 核心观点

- **两个独立的人 > 一个人格分裂的人** — 多实例独立 Agent 优于单实例多 Agent
- **双平台策略**：Railway ($5/月) + ClawCloud Run (~$0/月)
- Docker 容器 + 持久化存储 + noVNC 远程桌面
- 多渠道通信：Slack / Telegram / Discord
- 私有 Git 仓库作为 Agent 记忆备份
