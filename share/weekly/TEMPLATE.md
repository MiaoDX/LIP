## Style Rules

1. **Executive Summary**: 2-3 sentences max. Focus on "why this week matters."
2. **Critical Deadlines**: List only time-sensitive items (deadlines, registration closes, etc.).
3. **Key Highlights**: Group by domain. Each item = one line with tags + original source link.
4. **Tags**: Use `[VLN]`, `[Manipulation]`, `[Navigation]`, `[Simulation]`, `[Safety]`, `[Planning]`, `[Localization]`, etc.
5. **Relevance**: `[Very High Relevance]`, `[High Relevance]`, `[Medium Relevance]`, `[Low Relevance]`
6. **Original Source**: Every article MUST include the original Weekly Robotics link.
7. **Notable Events**: Keep the table format for events.
8. **Quick Take**: Synthesis, not summary. Connect dots between items. Include Action items.
