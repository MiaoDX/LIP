什么是 Tailscale？

零配置的私有网络

  • Tailscale 是基于 WireGuard 协议的零配置 VPN
  • 自动创建加密的私有网络 (tailnet)，让你所有设备互联互通
  • 无需端口转发、无需公网 IP、无需复杂配置
  • 设备之间直接点对点连接，延迟极低
  • 支持 macOS / Windows / Linux / iOS / Android

四种玩法

方法一：SSH + Tailscale + tmux Free

经典方案 — 完整开发环境

  • 在桌面电脑运行 Claude Code，拥有完整本地环境
  • Tailscale 创建私有网络，设备自动互联
  • 手机/平板通过 SSH 远程连接
  • tmux 保持会话活跃，断连后可恢复

操作步骤

  1. 两台设备都安装 Tailscale 并登录同一账号
  2. 在服务器上启用 Tailscale SSH：tailscale up --ssh
  3. 从移动设备连接：ssh user@machine-name
  4. 启动 tmux 并运行 Claude Code
# 服务器端
tailscale up --ssh
tmux new -s claude
claude

# 客户端（从手机/平板）
ssh user@my-desktop
tmux attach -t claude

方法二：Claude Code Remote Control Pro/Max

官方方案 — 最佳移动体验

  • 2026 年 2 月推出的 Research Preview
  • 将本地 Claude Code 桥接到 claude.ai/code、iOS、Android 应用
  • 完整本地环境可用，包括文件系统和工具
  • 需要 Claude Pro 或 Max 订阅

操作步骤

  1. 在本地启动远程模式：claude --remote
  2. 打开 claude.ai/code 或移动应用
  3. 自动连接到本地环境
# 在开发机器上
claude --remote

# 然后在任何浏览器或移动设备上
# 访问 claude.ai/code 即可连接

方法三：claude-code-tailscale MCP 插件 Free

多 Agent 协作 — 内网资源访问

  • 通过 tailnet 访问内部 API、Dashboard 等资源
  • 支持多 Agent 协调工作
  • 一键安装，即装即用
# 安装并使用
npx -y claude-code-tailscale

方法四：Tailscale + MCP Proxy Free

安全远程 MCP — 高级用法

  • 在远程机器上托管 MCP Server
  • MCP Client 指向 Tailscale 地址
  • 零公网暴露，完全通过 tailnet 加密传输
  • 适合需要访问生产环境数据的场景
# 远程机器（运行 MCP Server）
tailscale up
mcp-server --bind 0.0.0.0:8080

# 本地（配置 Claude Code MCP Client）
# 在 .claude/settings.json 中指向 tailscale 地址
"mcpServers": { "remote": { "url": "http://my-server:8080" } }

安全提示

使用 Tailscale ACLs 控制访问

配置访问控制列表，精确控制哪些设备可以访问哪些服务。只开放必要的端口和服务。

Auth Keys 设置过期时间

使用带过期时间的认证密钥。避免永久有效的 auth key，定期轮换提高安全性。

不要将 Claude Code 暴露到公网

Claude Code 应只通过 Tailscale 私有网络或本地访问，绝不应直接暴露到公共互联网。
